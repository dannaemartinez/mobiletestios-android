# Mobile tests

(You must have the localhost running)
To run the tests:

To run Android:

```
appium
```

```
node test_android.js
```

```
emulator -avd testing
```

To run iOS:

```
appium
```

```
node test_ios.js
```

```
open -a simulator
```

Remember to use the newest version of chrome:

```
 on the chrome navigator
```
